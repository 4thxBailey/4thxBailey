## Standards We Hold Ourselves To

Every tool we publish is:

- **Graph API v1.0 only** — no beta endpoints in production
- **Read-only** — no write, modify, or delete operations on client tenants
- **Cross-platform** — PowerShell 7.0+ on Windows, macOS, and Linux
- **Open and auditable** — MIT licensed, publicly versioned, CHANGELOG maintained

Our M365 security assessments reference **CISA Secure Configuration Baselines (SCuBA)**, **CIS Benchmarks**, **NIST SP 800-53**, and **MITRE ATT&CK**.
